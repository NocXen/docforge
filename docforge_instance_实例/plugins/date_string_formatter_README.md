# DateStringFormatter 日期字符串格式化器

## 概述

`DateStringFormatter` 是一个通用的transformer插件，用于将日期字符串转换为统一格式。支持多种输入格式（YMD、DMY、MDY）和丰富的输出格式，包括中文和英文日期格式。

## 功能特点

- 支持多种输入格式（YMD、DMY、MDY）
- 支持中文日期格式
- 支持英文日期格式（简写和完整）
- 支持自定义月份和日期名称
- 支持错误处理配置
- 自动验证日期有效性

## 配置参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| Target | string | 否 | - | 要格式化的日期字段名。如果未指定，跳过执行 |
| From | string | 否 | "YMD" | 输入日期格式。可选值：YMD、DMY、MDY |
| To | string | 否 | "{Y}年{M}月{D}日" | 输出日期格式模板 |
| Error | string | 否 | "!!DateFormattingError!! [{str}]" | 错误处理模板 |
| CustomM | dict | 否 | {} | 自定义月份名称映射 |
| CustomD | dict | 否 | {} | 自定义日期名称映射 |

## To格式占位符

| 占位符 | 说明 | 示例 |
|--------|------|------|
| {str} | 原值信息 | 2026/3/29 |
| {Y} | 四位年份 | 2026 |
| {YY} | 两位年份 | 26 |
| {M} | 数字月份（1~12） | 3 |
| {MM} | 两位补零月份（01~12） | 03 |
| {ensM} | 英文简写月份 | Mar |
| {enwM} | 英文完整月份 | March |
| {D} | 数字日期（1~31） | 29 |
| {DD} | 两位补零日期（01~31） | 29 |
| {ensD} | 英文简写日期 | 29th |
| {enwD} | 英文完整日期 | Twenty-ninth |

## 配置示例

### 中文日期格式
```json
{
    "Target": "日期",
    "From": "YMD",
    "To": "{Y}年{M}月{D}日",
    "Error": "!!DateFormattingError!! [{str}]"
}
```

### 英文简写格式
```json
{
    "Target": "日期",
    "From": "YMD",
    "To": "{ensM} {D}, {Y}",
    "Error": "!!DateFormattingError!! [{str}]"
}
```

### 英文完整格式
```json
{
    "Target": "日期",
    "From": "YMD",
    "To": "{enwM} {ensD}, {Y}",
    "Error": "!!DateFormattingError!! [{str}]"
}
```

### 自定义月份名称
```json
{
    "Target": "日期",
    "From": "YMD",
    "To": "{ensM} {D}, {Y}",
    "CustomM": {
        "9": "Sept",
        "11": "Nov."
    }
}
```

## 使用场景

### 场景1：统一日期格式

当你的数据源包含不同格式的日期时：

1. 使用 `excel_extractor` 提取日期数据
2. 使用 `date_string_formatter` 统一日期格式
3. 使用 `docx_replacer` 生成文档

### 场景2：生成英文报告

当你需要生成英文报告时：

1. 提取中文日期数据
2. 使用格式化器转换为英文格式
3. 生成英文报告

### 场景3：自定义日期显示

当你需要特殊的日期显示格式时：

1. 配置自定义月份名称
2. 使用格式化器生成特殊格式
3. 应用到文档中

## 支持的日期格式

### 输入格式（From）

- **YMD**: 年-月-日（如2026/3/29、2026-3-29、20260329）
- **DMY**: 日-月-年（如29/3/2026、29-3-2026）
- **MDY**: 月-日-年（如3/29/2026、3-29-2026）

### 输出格式（To）

- 中文格式：{Y}年{M}月{D}日
- 英文简写：{ensM} {D}, {Y}
- 英文完整：{enwM} {ensD}, {Y}
- 数字格式：{Y}-{MM}-{DD}
- 自定义格式：任意组合

## 日期验证

插件会自动验证日期的有效性：

- 月份范围：1-12
- 日期范围：1-31
- 每月天数验证
- 闰年二月验证

## 注意事项

1. **From格式**：如果未指定From格式，默认使用YMD
2. **To格式**：如果未指定To格式，默认使用中文格式
3. **错误处理**：可以通过Error参数自定义错误提示
4. **自定义名称**：CustomM和CustomD的优先级最高
5. **括号处理**：无效的括号内容会被自动删除

## 错误处理

插件会在以下情况返回错误模板：

- 日期字符串不包含3个数字
- 日期超出有效范围
- 月份超出1-12范围
- 日期超出当月最大天数
